# 俄羅斯方塊 (Tetris) 網頁遊戲實作計畫

這是一份為您開發純 HTML 俄羅斯方塊遊戲的計畫。為了達到「純 HTML」且「易於執行」的目標，我們將把 HTML 結構、CSS 樣式以及 JavaScript 邏輯整合在單一個 `index.html` 檔案中。

## 🎯 核心功能與設計

1. **視覺設計 (UI/UX)**
   - 使用 HTML5 `<canvas>` 來繪製遊戲主區域與「下一個方塊」的預覽區域。
   - 採用現代感深色主題 (Dark Mode) 搭配鮮豔的霓虹色彩方塊。
   - 顯示當前分數 (Score)、消除行數 (Lines) 等資訊。

2. **遊戲控制 (Controls)**
   - `⬅️ 左方向鍵`：向左移動
   - `➡️ 右方向鍵`：向右移動
   - `⬇️ 下方向鍵`：加速下落 (Soft Drop)
   - `⬆️ 上方向鍵`：順時針旋轉 (右旋)
   - `Z 鍵` 或 `Ctrl 鍵`：逆時針旋轉 (左旋)
   - `空白鍵 (Space)`：直接下落 (Hard Drop)

3. **遊戲邏輯 (Game Logic)**
   - 實作 7 種標準方塊 (I, J, L, O, S, T, Z) 及其專屬顏色。
   - 碰撞檢測：防止方塊超出邊界或重疊。
   - 消除判定：滿行自動消除並計算分數。
   - 預覽系統：顯示下一個即將出現的方塊。

4. **音效系統 (Sound Effects)**
   - 為了避免依賴外部音效檔案（導致需要額外下載或伺服器環境），我們將使用瀏覽器內建的 **Web Audio API** 來程式化合成復古的 8-bit 音效。
   - 包含的音效：移動、旋轉、方塊落地、消除行、遊戲結束。

> [!IMPORTANT]
> **需要您的審查 (User Review Required)**
> 1. **單一檔案設計**：為了方便，所有的 CSS 和 JS 都會放在 `index.html` 裡面。如果您希望將 CSS 和 JS 拆分成獨立檔案，請告知我。
> 2. **音效合成**：使用 Web Audio API 合成音效（嗶嗶聲等復古音效）是不需要額外檔案的最佳解，這點是否符合您的「要有音效」的期望？
> 3. **按鍵配置**：上箭頭為右旋轉，Z 鍵為左旋轉。這樣的「左右旋轉」配置是否可以，還是您希望兩者都綁在箭頭鍵上（這可能需要調整預設的下落按鍵）？

## 🧪 驗證計畫

### 手動驗證 (Manual Verification)
1. 在瀏覽器中打開 `index.html`。
2. 點擊畫面開始遊戲，測試左右箭頭移動是否順暢。
3. 測試上箭頭與 Z 鍵的旋轉是否正確。
4. 測試空白鍵能否「瞬間掉落」到底部。
5. 確認音效在各個操作下（移動、旋轉、消除等）能正常播放。
6. 確認「預覽下一個方塊」的視窗正確顯示。
